# 🏉 Application de Gestion Financière - Rugby Club Audomarois

## 🎯 À quoi sert cette application ?

Cette application a été créée spécialement pour le **Rugby Club Audomarois** afin de gérer facilement toutes les finances du club. Elle remplace les anciens fichiers Excel et permet de :

-   ✅ **Suivre l'argent** qui entre et sort du club
-   ✅ **Gérer les cotisations** des membres
-   ✅ **Enregistrer les dépenses** (matériel, repas, etc.)
-   ✅ **Avoir des comptes clairs** et transparents
-   ✅ **Générer les attestations fiscales** automatiquement
-   ✅ **Sécuriser les données** avec des droits d'accès

---

## 💰 Les Comptes du Club

Le club gère **2 types de comptes** :

### 🏦 Compte Courant

-   C'est le compte principal du club
-   Toutes les cotisations, dons, achats y sont enregistrés
-   Solde visible en temps réel

### 💰 Livret d'Épargne

-   Réserve d'argent mise de côté
-   Permet de faire des économies pour les gros projets
-   Transferts possibles avec le compte courant

## 📊 Organisation des Données

### 📅 **Exercices Financiers**

Chaque saison sportive = 1 exercice financier

-   Exemple : "Saison 2024-2025"
-   Période définie (septembre à août)
-   Peut être fermé en fin de saison pour figer les comptes

### 💳 **Transactions**

Chaque mouvement d'argent est une transaction :

-   **Entrées d'argent** (cotisations, dons, subventions...)
-   **Sorties d'argent** (achats, frais, remboursements...)
-   Chaque transaction a un numéro d'ordre automatique
-   Reliée à un membre ou une entreprise

### 🏷️ **Types de Transactions**

Catégories prédéfinies :

-   **Cotisation** (seulement entrées d'argent)
-   **Don** (seulement entrées d'argent)
-   **Achat matériel** (seulement sorties d'argent)
-   **Repas amicale** (entrées ou sorties selon le cas)

## 🔐 Sécurité et Confidentialité

### 🛡️ **Protection des données**

-   Mots de passe cryptés
-   Accès sécurisé par login/mot de passe
-   Respect du RGPD (consentement, droit à l'oubli)

### 👮 **Contrôle des droits**

-   Chaque utilisateur a des permissions spécifiques
-   Impossible d'accéder aux fonctions non autorisées
-   Traçabilité de toutes les actions

### 💾 **Sauvegarde**

-   Toutes les données sont sauvegardées automatiquement
-   Aucun risque de perte (contrairement aux fichiers Excel)
-   Historique complet conservé

## 📱 Utilisation au Quotidien

### 🏠 **Tableau de Bord**

Dès la connexion, vous voyez :

-   💰 Solde total du club
-   🏦 Solde du compte courant
-   💰 Solde du livret d'épargne
-   📊 Évolution des comptes

### ➕ **Ajouter une Transaction**

1. Cliquer sur "Nouvelle Transaction"
2. Remplir les informations :
    - Description (ex: "Cotisation Jean Dupont")
    - Montant (ex: 25,00 €)
    - Membre concerné
    - Type de transaction
    - Mode de paiement
3. Valider → Le solde se met à jour automatiquement

### 📄 **Générer une Attestation Fiscale**

1. Aller dans "Attestations Fiscales"
2. Choisir le membre
3. Sélectionner la période
4. Télécharger le PDF généré automatiquement
