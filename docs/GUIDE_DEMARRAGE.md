# 🚀 Guide de Démarrage Rapide - Application RCA

## ⚡ En 5 minutes, comprenez l'essentiel !

### 🎯 **L'application en une phrase**

Cette application remplace vos fichiers Excel pour gérer l'argent du Rugby Club Audomarois de manière sécurisée, automatique et transparente.

---

## 🏆 **Pourquoi cette application ?**

| ❌ **AVANT (Excel)**   | ✅ **MAINTENANT (Application)** |
| ---------------------- | ------------------------------- |
| Fichiers perdus        | Sauvegarde automatique          |
| Erreurs de calcul      | Calculs fiables                 |
| Pas de droits d'accès  | Sécurité par utilisateur        |
| Difficile à partager   | Accessible partout              |
| Attestations manuelles | PDF générés automatiquement     |

---

## 👥 **Qui fait quoi ?**

```
🌍 Visiteur          → Voir la page d'accueil, se connecter
👤 Membre           → Voir les soldes + ses données
💼 Trésorier        → Gérer toute la comptabilité
🔧 Administrateur   → Configuration du système
```

## 🎪 **Actions courantes**

### 💳 **Ajouter une cotisation**

1. Se connecter en tant que trésorier
2. "Nouvelle Transaction"
3. Remplir : membre, montant, type "cotisation"
4. Valider → Solde mis à jour automatiquement

### 📄 **Générer une attestation fiscale**

1. Aller dans "Attestations"
2. Choisir le membre et l'année
3. Télécharger le PDF généré

### 📊 **Voir les comptes**

1. Se connecter
2. Tableau de bord → Tous les soldes affichés
3. Historique complet disponible
